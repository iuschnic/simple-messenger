# Резюме проекта: Min Messenger

## Цель и ожидаемые выгоды
- Завершить курс «Командная разработка»
- Получение практического опыта, пополнение портфолио
- Получение инструмента для внутренних коммуникаций

## Ожидаемые стоимость и срок реализации
- **Срок реализации:** 2 месяца  
- **Бюджет:** 0 руб. (допускаются минимальные расходы на облачную инфраструктуру — порядка 500–1000 руб./мес)

## Требования и ожидания по качеству
- Сообщения должны доставляться получателю без потерь
- Система не должна терять данные при перезапуске
- Интерфейс должен быть интуитивно понятным
- Подходит как для личного общения, так и для работы в командах
